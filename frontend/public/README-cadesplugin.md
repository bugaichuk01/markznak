# cadesplugin_api.js

Файл `cadesplugin_api.js` — официальная обёртка КриптоПро для браузера.

- Источник: https://www.cryptopro.ru/sites/default/files/products/cades/cadesplugin_api.js
- Подключается в `index.html` **до** React-бандла.
- Создаёт `window.cadesplugin` как **Promise**; в коде нужно `await window.cadesplugin`.

Обновление: скачайте свежую версию с сайта ЦРПТ или сохраните из DevTools → Sources со страницы песочницы СУЗ.
